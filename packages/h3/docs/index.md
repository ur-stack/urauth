# @urauth/h3

H3/Nitro middleware for urauth. Adds JWT verification, permission guards, and auth context to H3 event handlers and Nuxt server routes.

## Installation

```bash
pnpm add @urauth/h3
```

## Peer Dependencies

- `h3` >= 1.0.0
- `@urauth/node`
- `@urauth/ts`

::: info
This package is under active development. Documentation will be expanded as the API stabilizes.
:::
