# @urauth/fastify

Fastify plugin for urauth. Adds JWT verification, permission guards, and auth context to Fastify routes.

## Installation

```bash
pnpm add @urauth/fastify
```

## Peer Dependencies

- `fastify` >= 4.0.0
- `fastify-plugin` >= 5.0.0
- `@urauth/node`
- `@urauth/ts`

::: info
This package is under active development. Documentation will be expanded as the API stabilizes.
:::
