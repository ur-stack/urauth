# @urauth/express

Express middleware for urauth. Adds JWT verification, permission guards, and auth context to Express routes.

## Installation

```bash
pnpm add @urauth/express
```

## Peer Dependencies

- `express` >= 4.0.0
- `@urauth/node`
- `@urauth/ts`

::: info
This package is under active development. Documentation will be expanded as the API stabilizes.
:::
