# @urauth/hono

Hono middleware for urauth. Adds JWT verification, permission guards, and auth context to Hono routes.

## Installation

```bash
pnpm add @urauth/hono
```

## Peer Dependencies

- `hono` >= 4.0.0
- `@urauth/node`
- `@urauth/ts`

::: info
This package is under active development. Documentation will be expanded as the API stabilizes.
:::
